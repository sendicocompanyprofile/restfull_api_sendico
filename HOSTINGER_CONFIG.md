# Hostinger Cloud Storage Configuration

Untuk menggunakan Hostinger cloud storage, Anda perlu:

1. **Hostinger Control Panel Credentials:**
   - API Token dari Hostinger
   - atau FTP credentials

2. **Opsi Implementasi:**
   - Option A: Hostinger API (if available)
   - Option B: S3-compatible endpoint (recommended)
   - Option C: FTP Upload

**Untuk sekarang, kami gunakan S3-compatible approach** (Hostinger support ini).

Butuh info:
- Hostinger S3 Endpoint
- Access Key ID
- Secret Access Key
- Bucket Name

Atau jika Anda punya FTP, kami bisa gunakan FTP approach.

File ini akan diupdate dengan credentials configuration.
